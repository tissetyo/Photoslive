# Privacy, retensi, dan disclosure pembayaran

Photoslive meminta consent berversi sebelum sesi. Foto dipakai untuk capture,
frame/GIF, print, upload, dan halaman hasil. Link hasil memakai kode acak dan
berakhir maksimal 24 jam. Pelanggan dapat menghapus sesi lebih awal; metadata dan
object terkait masuk cleanup terautentikasi.

Foto yang belum berhasil di-upload tidak dihapus oleh age retention. Akses file
selalu memvalidasi sesi, expiry, dan manifest. Transport serta storage provider
wajib terenkripsi; log menyensor token, cookie, credential, signed URL, email,
dan rekening sensitif.

QRIS membutuhkan internet. Sebelum membayar, UI wajib menampilkan nominal,
currency IDR, produk (sesi/print), status pending/paid/expired/refunded, dan
provider. Payment tidak menjamin printer berhasil; foto tetap tersedia dan
operator menangani retry/refund sesuai kebijakan booth. Platform fee, provider
fee, payout schedule, dan status KYC harus terlihat kepada owner sebelum fitur
finance produksi diaktifkan.

Dokumen ini adalah disclosure produk teknis, bukan pengganti terms/legal review.
Production memerlukan approval privacy dan finance/compliance terpisah.
